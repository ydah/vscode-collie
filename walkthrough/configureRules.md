Create a `.collie.yml` in the workspace root to keep linting and formatting settings with the grammar project.

Use [Collie: Create .collie.yml](command:collie.createConfig) to create the file, or [Collie: Open Collie Config](command:collie.openConfig) to jump to the configured path.

If the config lives somewhere else, set `collie.configPath` in VS Code settings.
