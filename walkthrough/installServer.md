Collie uses the `collie-lsp` Ruby gem for diagnostics, formatting, quick fixes, and navigation.

```bash
gem install collie-lsp
```

After installing, run [Collie: Check Setup](command:collie.checkSetup). If your workspace uses Bundler, enable `collie.useBundler` or set `collie.lspPath` to the executable you want VS Code to run.
